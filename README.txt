EdTech Platform - Quick Start Guide
==================================

This is a demo EdTech platform for student and teacher dashboards, AI reports, and profile management.

How to Run:
-----------
1. Make sure you have Node.js and npm installed on your computer.
2. Open a terminal and navigate to the project directory.
3. Install dependencies:
   npm install
4. Start the development server:
   npm run dev
5. Open your browser and go to:
   http://localhost:3000

Features:
---------
- Student and teacher dashboards
- AI-generated reports
- Profile and profile settings pages
- Account management (edit profile, change password, delete account)

Notes:
------
- The system simulates user accounts and student data as if there is a database in the background.
- All changes (profile edits, deletions) are for demo purposes and are not persisted after a server restart.
- To log in as a student, use one of the provided mock email addresses and the password '123456'.

For more details, explore the dashboard and profile sections in your browser.
